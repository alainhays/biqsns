Get class name which belong to attributes only, by remove first 2 classes which is default to be exist ( 'biq-widgets' and a class name of widget )

Parameters

Returns
{String} contain class filtered ( 2 default class removed )
BIQWidgetElementParser prototype getDefaultValue

Important to get default value of 'key'.

Usually necessary when default input can be empty to load based on default value.

Parameters
[Object]p_structure_item structure of widget as defined on file biq-widget-structure.js
{String}p_key key of input / value type
Returns
{String} the default value
BIQWidgetElementParser prototype getValues

Main function to Get values based on widget name by call other functions.

Parameters
{String}p_widget_type is based on widget type wich is belong to html attribute named data-biq-widget-type which need to convert to function name which is using camel case convention ( by using typeToFunction() function)
{Object}p_widget_sel HTML element got from jQuery function $('#id'). It is selected widget by click 'Edit' after hovering
{Object}p_structure_sel Selected structure by matching p_widget_sel and defined at biq-widget-structure.js.
Returns
{Object} generated by function called with self[p_widget_type] which give reference to a function
