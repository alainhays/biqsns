Detailed Description

For updating widget data of theme.

This class is mainly purposed for updating widget which stored at Wordpress 'option' table

Member Function Documentation

BIQTheme::save (   $p_arr)

Save modified widget with wordpress function : update_option()

This function should be called via AJAX at file backend/ajax-submit.php by using wp-ajax-. It will first detect 'widget_type' from $_POST variable then call appropriate function related to widget type. e.g : contact_email_simple_update()

Parameters
array$p_arrgot from $_POST PHP variable contain 'widget_type' and other widget detail
